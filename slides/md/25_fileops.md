## File Operations

1. Write mode 	: 'w' (overwrites)
2. Append mode	: 'a'
3. Read mode		: 'r'
4. Write/Append creates file if necessary
5. Open('filename') returns a **file handle**
6. Use this handle to write to/read from file
7. file\_handle.close()
8. Shorthand : **with...as**
9. **f.read()** vs **f.readline()**
